# 背單字 Vocab App

一個簡單的單字學習網站：可以貼上大量單字、上傳單字表檔案，或上傳照片自動辨識英文單字，
系統會自動查詢中文意思，並可以用多種方式做測驗。

## 功能

- **新增單字的三種方式**
  1. 貼上/手動輸入：可以一次貼上一大串單字（每行一個「英文 中文」，或用逗號、空白、換行隔開的一堆英文單字都可以），系統會自動辨識格式並排版整理。
  2. 上傳單字表檔案（.txt / .csv）。
  3. 上傳照片：用 OCR（Tesseract）自動辨識照片中的英文單字。
- 沒有中文意思的單字，會自動用免費翻譯 API 查詢；查不到的會標示「需要確認」，可在預覽畫面手動補上或之後在單字庫直接點兩下修改。
- 單字可以分類（例如 Unit 1、TOEIC），測驗時可以選擇只測某個分類。
- 測驗模式：
  - 英文 → 中文（選擇題）
  - 中文 → 英文（選擇題）
  - 拼字測驗（看中文意思，打出正確英文拼字）
  - 聽力測驗（用瀏覽器發音朗讀單字，聽完拼出來）
- 系統會記錄每個單字答對/答錯次數，測驗出題時會優先挑選比較不熟的單字。

## 安裝

1. 安裝 Tesseract OCR（照片辨識需要）：
   - macOS: `brew install tesseract`
   - Ubuntu/Debian: `sudo apt-get install tesseract-ocr`
   - Windows: 至 [tesseract 官方頁面](https://github.com/UB-Mannheim/tesseract/wiki) 下載安裝，並確認 `tesseract` 指令可在終端機執行。

2. 安裝 Python 套件：

   ```bash
   cd vocab_app
   pip install -r requirements.txt
   ```

3. 啟動網站：

   ```bash
   python app.py
   ```

   預設在 http://127.0.0.1:5000 開啟，資料會存在同資料夾底下的 `vocab.db`（SQLite，第一次啟動會自動建立）。

## 注意事項

- 自動查詢中文意思的功能需要連上網路（使用 [MyMemory](https://mymemory.translated.net/) 免費翻譯 API）。如果目前網路無法連線或查詢失敗，該單字會標示「需要確認」，可以自行輸入中文意思，或之後在單字庫頁面點兩下該欄位修改。
- 照片 OCR 辨識準確度取決於照片畫質、字體與排版，建議拍攝時光線充足、單字清楚、避免手寫斜體，辨識完成後務必在預覽畫面確認一次再儲存。
- 聽力測驗使用瀏覽器內建的語音合成（Web Speech API），不需要額外安裝，但音質與可用語音依瀏覽器/作業系統而異。
