# 背單字 Vocab App

一個功能完整的個人單字學習網站：貼上/上傳/拍照新增單字、自動查中文意思、多種測驗與複習模式，
並用間隔重複（Spaced Repetition）系統排定複習進度。

## 功能總覽

### 新增單字（三種方式，都支援「一次貼上大量單字自動排版」）
1. **貼上/手動輸入**：一次貼上一大串單字，系統自動辨識格式並排版整理，支援：
   - 每行「英文 中文」，分隔可以是空白、全形空白、冒號、逗號、破折號，甚至完全沒有分隔（`apple蘋果`）
   - 純英文單字列表，用逗號、空白、換行、編號（`1. apple`）都可以混著貼
2. **上傳單字表檔案**（.txt / .csv）
3. **上傳照片辨識**：用 Tesseract OCR 自動辨識照片中的英文單字

沒有中文意思的單字會自動查詢翻譯（[MyMemory](https://mymemory.translated.net/) 免費 API），查不到的會標示「需要確認」，可在預覽畫面或單字庫（點兩下欄位）手動補上。

### 單字庫管理
- 搜尋（英文/中文）、依分類（單字表）篩選、依「已精熟／學習中／已加星號」篩選
- 星號標記重要單字
- 多選批次刪除 / 批次搬移到其他分類
- 匯出 CSV
- 每個單字可點喇叭圖示聽發音（瀏覽器 TTS）

### 間隔重複複習系統（Leitner box）
- 每個單字有 1~5 盒的熟悉度：答對晉級（複習間隔變長：0 / 1 / 3 / 7 / 14 天），答錯退回第 1 盒
- 總覽頁面顯示熟悉度分布圖、今日待複習數量、整體正確率

### 複習 / 測驗模式
- 🔁 **單字卡複習**：翻卡片、「還不熟 / 我記得」，支援鍵盤（空白鍵翻卡、← →作答），會優先出到期複習的單字
- 📝 **測驗**：英文→中文選擇題、中文→英文選擇題、拼字測驗（看中文寫英文）、聽力測驗（發音朗讀後拼字），可選擇只考今天到期的單字
- 🧩 **配對遊戲**：類似 Quizlet 的配對玩法，計時 + 計算配對錯誤次數

### 介面
- 響應式版面（手機/電腦皆可用），支援深色模式（依系統設定自動切換，也可手動用右上角按鈕切換，會記住偏好）

## 安裝與啟動

### 最簡單：一鍵腳本

下載這個專案後，在 `vocab_app` 資料夾執行：

- **macOS / Linux**：
  ```bash
  cd vocab_app
  ./start.sh
  ```
- **Windows**：直接雙擊 `vocab_app/start.bat`（或在 cmd 執行）。

腳本會自動檢查/安裝 Tesseract OCR、建立 Python 虛擬環境、安裝套件，並啟動網站在 http://127.0.0.1:5000。
之後要再次啟動，一樣執行同一個指令即可（套件已裝好的話會很快）。

### 手動安裝（如果不想用腳本）

1. 安裝 Tesseract OCR（照片辨識需要）：
   - macOS: `brew install tesseract`
   - Ubuntu/Debian: `sudo apt-get install tesseract-ocr`
   - Windows: 至 [tesseract 官方頁面](https://github.com/UB-Mannheim/tesseract/wiki) 下載安裝，並確認 `tesseract` 指令可在終端機執行。

2. 安裝 Python 套件：

   ```bash
   cd vocab_app
   pip install -r requirements.txt
   ```

3. 啟動網站：

   ```bash
   python app.py
   ```

   預設在 http://127.0.0.1:5000 開啟，資料會存在同資料夾底下的 `vocab.db`（SQLite，第一次啟動會自動建立）。

## 注意事項

- 自動查詢中文意思的功能需要連上網路。如果查詢失敗，該單字會標示「需要確認」，可以自行輸入中文意思，或之後在單字庫頁面點兩下該欄位修改。
- 照片 OCR 辨識準確度取決於照片畫質、字體與排版，建議拍攝時光線充足、單字清楚，辨識完成後務必在預覽畫面確認一次再儲存。
- 發音功能使用瀏覽器內建的語音合成（Web Speech API），不需要額外安裝，但音質與可用語音依瀏覽器/作業系統而異。
- 這是單人使用的本機小工具，沒有帳號登入機制；`vocab.db` 就是你所有的資料，記得備份。
